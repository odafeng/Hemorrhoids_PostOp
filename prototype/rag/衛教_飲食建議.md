# 飲食建議

## 總原則

術後飲食的目標是**軟化糞便、促進排便順暢**，避免硬便或用力排便造成傷口出血和疼痛。

## 術後飲食三大原則

### 1. 高纖維飲食
- **每日建議**：膳食纖維 25-30 克
- **蔬菜**：地瓜葉、空心菜、菠菜、花椰菜、木耳
- **水果**：木瓜、香蕉、奇異果、火龍果、柳橙
- **全穀類**：糙米、燕麥、全麥麵包
- **豆類**：毛豆、黑豆、紅豆（適量）
- **注意**：纖維攝取要循序漸進增加，突然大量攝取可能會脹氣

### 2. 充足水分
- **每日建議**：至少 2000-2500 毫升（約 8-10 杯水）
- 高纖維飲食必須配合足夠水分，否則反而會便秘
- 可以喝白開水、淡茶、果汁
- 避免含咖啡因飲料過量（咖啡、濃茶有利尿效果）

### 3. 適量油脂
- 適當的油脂有助於潤滑腸道
- 橄欖油、苦茶油、堅果類都是好選擇
- 不需要刻意低脂飲食

## 建議食物

| 類別 | 推薦食物 |
|------|---------|
| 早餐 | 燕麥粥、全麥吐司、優格加水果 |
| 主食 | 糙米飯、五穀飯、地瓜 |
| 蔬菜 | 深綠色蔬菜、菇類、海帶 |
| 水果 | 木瓜、香蕉（熟的）、奇異果 |
| 湯品 | 蔬菜湯、排骨蘿蔔湯 |
| 點心 | 堅果、黑棗、無花果乾 |

## 應避免的食物

### 刺激性食物（術後 2-4 週內避免）
- **辣椒、麻辣鍋、咖喱**：刺激腸道和肛門黏膜
- **酒精**：擴張血管，增加出血風險
- **濃咖啡**：可能加重腹瀉或刺激腸道

### 容易便秘的食物
- 精緻白米、白麵條（低纖維）
- 過多肉類而缺少蔬菜
- 加工食品、速食
- 柿子、芭樂（未熟的、含鞣酸）

### 容易產氣的食物（適量即可）
- 豆漿、牛奶（乳糖不耐者）
- 大量高麗菜、洋蔥
- 碳酸飲料

## 術後第一天飲食

- 手術當天可以正常飲食，不需要禁食
- 建議先從**清淡、軟質**食物開始
- 粥、湯麵、蒸蛋、豆腐都是好的選擇
- 之後逐漸恢復正常飲食

## 常見問題

**Q: 術後可以喝咖啡嗎？**
A: 少量可以（1 天 1 杯），但要額外補充水分。避免空腹喝咖啡，也不要喝太濃。

**Q: 可以吃辣嗎？**
A: 建議術後 2-4 週內避免辛辣食物。辣椒素會刺激肛門黏膜，可能加重疼痛和不適。傷口癒合後可以逐漸恢復。

**Q: 需要吃什麼保健品嗎？**
A: 一般不需要特別補充。如果飲食中纖維不足，可以考慮使用纖維補充品（如車前子殼粉 psyllium）。請先諮詢醫師。

**Q: 素食者怎麼辦？**
A: 素食者通常纖維攝取已經足夠，但要注意蛋白質的攝取（豆腐、豆類）幫助傷口癒合，以及充足水分。
